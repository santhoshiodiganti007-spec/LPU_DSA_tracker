# Longest Consecutive Sequence

https://leetcode.com/problems/longest-consecutive-sequence/submissions/2053297120/
```
class Solution:
    def longestConsecutive(self, nums: List[int]) -> int:
        numSet = set(nums)
        longest = 0

        for num in numSet:
            if num - 1 not in numSet:
                current = num
                count = 1

                while current + 1 in numSet:
                    current += 1
                    count += 1

                longest = max(longest, count)

        return longest

```
